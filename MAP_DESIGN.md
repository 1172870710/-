## 三、中央广场 — 详细设计

**尺寸：** 直径 300px（约 9-10 格）
**位置：** 地图水平居中，纵向略偏上（给农场和森林更多空间）
**铺装：** 同心圆砖石纹路
- 外圈：深灰色石板（环形排列）
- 中圈：浅灰色石板
- 内圈：水蓝色渐变瓷砖

### 中心喷泉
- **结构：** 三层递进式圆台（大→中→小）
  - 底层：直径 80px 的八角形浅水池
  - 中层：直径 50px 的石台，刻有波浪纹
  - 顶层：直径 20px 的小台，中心出水
- **水柱：** 最高 ~60px，像素风透明蓝白渐变
- **池水：** 清澈见底，可见浅池底的鹅卵石纹

### 周边元素
- 8 根路灯：等距分布在广场圆周上，黑色铸铁造型，暖黄色光晕
- 4 条长椅：棕色木质，放置在十字路口的对角线方向
- 广场边缘：低矮石柱（每 40px 一根）+ 铁链相连（装饰性围栏）

### 道路接口
| 方向 | 宽度 | 路面 | 通往 |
|------|------|------|------|
| 北 | 64px | 石板路 | 住宅区 Y 形岔路 |
| 南 | 64px | 石板路 | 农场区 |
| 东 | 64px | 石板路 | 商业区 |
| 西 | 48px | 石板路 | 住宅区西侧 |
| 东南 | 40px | 鹅卵石路 | 森林入口 |

### 玩家出生点
- 位置：广场正南方，距广场边缘约 64px
- 地面：浅色石板小平台
- 装饰：左右各一棵遮阴树
- 旁边木牌："欢迎来到平安镇"

### 隐藏元素
- **Phase 2：** 广场东北角有一块颜色略微不同的地砖 → 可撬开 → 下面是空的暗格（曾藏过某物）
- **Phase 3：** 喷泉底部有一块可活动的石砖 → 旋转 → 水池底部打开 → 阶梯通往地下设施

## 四、NW 住宅区 — 详细设计

**范围：** 广场西北方向，三条小路呈 Y 形延伸
**路面：** 暖色石板小路（32-40px 宽），弯曲自然
**绿化：** 稀疏落叶乔木，住户门口有小花坛

| 角楼 | 西南角突出一座 40×40 的方形角楼，比主楼高半层 |
| 角楼顶 | 带锥形小尖顶，顶上有一枚风信鸡 |
| 门廊 | 正面宽门廊（40×16px），两根圆木柱支撑 |
| 正门 | 深色双开木门，铁质合页外露 |
| 招牌 | 门口右侧，木牌"平安旅馆"，悬挂一枚红灯笼 |

## 七、SE 森林区 — 详细设计

**氛围：** 深绿色草地 + 密集树木，越往深处越暗
**入口：** 从广场东南鹅卵石路进入，宽 40px

### ⑧ 林中木屋（涛）

| 项目 | 规格 |
|------|------|
| 占地 | 100×80px（极不规则平面） |
| 墙体 | 粗糙深褐色木板 + 树皮混合，参差不齐 |
| 屋顶 | 拼凑材料：左半边木板、右半边铁皮、中间树皮 |
| 贯穿的树 | 一棵粗树干从门廊右侧穿过屋檐伸出 |
| 正门 | 一半的木门，半掩着，门槛上长蘑菇 |
| 窗户 | 大小不一：一个 20×20 方窗、一个 12×16 竖窗、一个圆洞 |
| 鹿角 | 门框上方挂一副鹿角（灰白色） |
| 风干肉 | 屋檐下垂着几条风干肉/兽皮 |
| 柴堆 | 门口左侧，乱堆的树枝 |
| 陷阱 | 屋前地面有几个兽夹（画出来警示性） |
| 酒瓶 | 窗台上倒着几个空酒瓶 |

### 森林
- **密度：** 中密到高密，越往东南越密集
- **树木：** 深绿/浅绿圆形树冠交替，树干棕色
- **林间小径：** 土黄色蜿蜒小路，宽 32px
- **蘑菇/浆果：** 林间零星点缀

### 湖泊
- **位置：** 森林中部偏西
- **大小：** ~200×150px
- **形状：** 不规则天然湖泊
- **颜色：** 深蓝色水体，边缘浅滩
- **装饰：** 岸边一两株垂柳 + 浮萍 + 芦苇

**出口位置：** 森林最东南角，地图边缘
- **Phase 1：** 看起来是一条延伸到远方的路，被雾气遮挡
- **Phase 2：** 走近发现路被一堵不可翻越的铁丝网挡住了，铁丝网上挂着一个"危险"标志
  - 涛：那条路啊？我试过，走不通的。也就骗骗新来的人。
- **Phase 3：** 找到铁丝网的缺口 → 爬过去 → 穿过一段浓雾 → 到达一个空旷的白房间（实验设施的"出口厅"）
  - 白房间有一扇门，门上写着："实验体转换区 - 切片重组将在跨越后自动进行"

**隐藏元素：**
- **Phase 2：** 涛的床底下有一个铁皮箱 → 里面装着他收集的"前任外来者留下的东西"
  - 包括：一本写满密码的笔记本、一枚标着"CTRL-#004"的金属徽章、一张被撕碎又粘好的地图（标注了设施的大致位置）
- **Phase 3：** 涛的木屋地下有一个深度约 5m 的枯井 → 井底侧面有一个铁门 → 通往设施的后勤通道

## 八、隐藏元素时间线总结

| 区域 | Phase 1 (温馨) | Phase 2 (发现) | Phase 3 (真相) |
|------|---------------|---------------|---------------|
| 广场 | 正常的喷泉广场 | 发现特殊地砖暗格 | 喷泉底暗门→地下设施 |
